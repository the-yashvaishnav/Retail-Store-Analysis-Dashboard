# Retail-Store-Analysis-Dashboard

## 📌 Description
This project presents an interactive dashboard to analyze retail store data, including sales performance, product categories, customer purchasing behavior, and store-wise trends. It helps in identifying top-selling products, sales patterns, and key business insights to support data-driven decision-making.

## 📊 Tools Used
- Tableau

## 📈 Key Insights
- Total Sales
- Total Products 
- Category and Sub-Category wise Sales Distribution
- Monthly Sales Trend
- Region-wise Profit Distribution
